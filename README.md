# Pythonで作ったテトリスゲーム

ご覧いただきありがとうございます。日本人のPython初心者が作ったテトリスゲームです。

## 環境

- Windows 10 64 bit
- Python 3.7.3
- pygame 1.9.5

## 遊び方

pythonとpygameをインストールしてください。

main.pyを実行するとスタートです。

ブロックが横に一列揃ったらそのブロックは消えます。ブロックが画面の一番上に当たったらゲームオーバーです。一列ブロックを消すごとに、ブロックが落ちるスピードは速くなります。

ゲームオーバーの後、再度プレイするには一度終了して起動してください。

## 操作方法

- →：右移動
- ←：左移動
- ↑：回転
- ESC：終了
